Convert EDU Links to HD Head URLS

by xTeal

Site where you can convert Minecraft education (image) links to JSON. For some guy named Panther.
WARNING: As of 7/18/21, education.minecraft.net (to the best of my knowledge) has removed the function of uploading files. Your HD Heads will NOT work at all. For any existing HD Heads please archive your usercache.json file and /assets/skins/ folder. This will cache the texture for them.

If you need a template, see here (4096x4096 img)

Input Field

Shove in links above. Separate it by new lines. To add a name to the item, format it like: link name (links can't have spaces, so it SHOULD work.)
